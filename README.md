# Work-Notes
## Kali Linux ssh
開啟
```
$ sudo service ssh start
```
查詢是否開啟
```
$ netstat -ap | grep ssh
```

## Putty ssh root 登入
vim /etc/ssh/sshd_config
```
PermitRootLogin yes
```
